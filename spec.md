# Specification

## Summary
**Goal:** Build a single-page frontend pitch deck and app demo for FreshGuard Inc., styled for a Shark Tank investor audience.

**Planned changes:**
- Hero section with FreshGuard logo, slogan "Stay Fresh. Stay Smart. Waste Nothing.", and a CTA button using the green color scheme (#2ECC71, #27AE60)
- Company info & mission section with mission statement and three team members (CEO Jordan Ellis, CTO Maya Chen, COO Marcus Webb)
- "How It Works" section with 4 sequential steps illustrated by emoji icons covering the device's core features
- Product features showcase with 5 feature cards (active carbon filter, ethylene gas sensor, barcode scanner, Bluetooth notifications, auto-mist cleanser) using emoji icons in a grid layout
- Target market section with demographics (ages 25–50, health-conscious families/busy professionals) and unique value proposition
- Shark Tank-themed financials section with dramatic dark styling: $500K ask for 15% equity, $3.3M valuation, COGS/retail pricing, monthly expense breakdown, current sales, and Year 1–2 revenue projections
- Interactive app demo section displaying a phone frame mockup with a color-coded food inventory list (green/yellow/red), alerts panel with sample warnings, and recipe suggestions for expiring items
- Scroll-triggered entrance animations for all sections, hover effects on cards, and a dramatic entrance animation for the financials section
- Fully responsive layout across desktop and mobile, cohesive green/white/charcoal visual theme with modern sans-serif typography and consistent emoji usage

**User-visible outcome:** Visitors see a polished, investor-ready single-page pitch site for FreshGuard Inc. that presents the product, team, financials, and an interactive app demo in one scrollable experience.
