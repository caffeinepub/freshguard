# Specification

## Summary
**Goal:** Add a Partners section to the FreshGuard landing page and implement a slide-in sidebar navigation drawer accessible from all pages.

**Planned changes:**
- Add an "Our Partners" section on the home/landing page displaying Frigidaire, Whirlpool, and KitchenAid as partner brands with their logos, placed after the Hero or Features section, styled to match the FreshGuard brand
- Add a persistent hamburger/menu button visible on all pages that opens a sidebar navigation drawer
- Sidebar drawer contains navigation links: Business Plan, How It Works, Financials, The Pitch, and Sharks Vote
- Clicking a sidebar link navigates/scrolls to the corresponding section and closes the sidebar
- Clicking outside the sidebar closes it
- Sidebar is styled consistently with the FreshGuard brand in a clean, minimal style

**User-visible outcome:** Visitors can see FreshGuard's three brand partners on the landing page, and can open a sidebar from any page to quickly navigate to any section of the site.
